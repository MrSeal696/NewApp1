Проект на **.NET MAUI**, в котором реализованы функционал управления задачами с локальным хранением, API и полноценной навигацией.

---

## 🔹 Описание проекта

Приложение позволяет:

* Создавать, редактировать и удалять задачи
* Просматривать список задач
* Сохранять данные локально (SQLite)
* Синхронизировать задачи с облачным API
* Просматривать отчёты и статистику
* Использовать кроссплатформенные возможности (папка приложения, уведомления, API)

Архитектура приложения построена по паттерну **MVVM**.

---

## 🔹 Структура проекта

* **Models** – модели данных (TaskItem)
* **ViewModels** – логика и команды
* **Pages** – страницы приложения (TaskListPage, AddTaskPage, ReportsPage, SettingsPage)
* **Services** – работа с данными и API
* **AppShell** – навигация по приложению

---

## 🔹 Технологии

* .NET MAUI (C#)
* SQLite для локального хранения
* HttpClient для работы с REST API
* MVVM для разделения логики и UI
